# Демонстрационный стенд модуля `hashlib`

## 1. Общая информация

**Название проекта:** Демонстрационный стенд модуля `hashlib`  
**Заказчик:** Учебная организация HERNY  
**Исполнитель:** Группа HERNY  

**Цель проекта:**  
Создать консольное приложение для наглядной демонстрации возможностей модуля `hashlib` стандартной библиотеки Python.

---

## 2. Описание задачи

Необходимо разработать консольное Python‑приложение, которое:

- показывает теоретическую информацию о хеш‑функциях и модуле `hashlib`;
- демонстрирует работу разных алгоритмов хеширования (MD5, SHA1, SHA256, SHA512);
- позволяет пользователю:
  - хешировать строки;
  - сравнивать хеши двух строк;
  - хешировать файлы;
  - хешировать сразу несколько строк.

Программа должна иметь удобное текстовое меню, в котором пользователь выбирает нужную демонстрацию по номеру.

---

## 3. Функциональные требования

1. **Меню приложения**
   - Отображение главного меню при запуске.
   - Выбор пунктов по цифрам (`0–8`).
   - Возврат в меню после завершения демонстрации.

2. **Теоретические разделы**
   - Введение: что такое хеш и модуль `hashlib`.
   - Обзор основных алгоритмов: `md5`, `sha1`, `sha256`, `sha512`.
   - Объяснение разницы между `digest()` и `hexdigest()`.
   - Краткие заметки по безопасности (где можно и где нельзя использовать обычные хеш‑функции).

3. **Практические демонстрации**
   - Хеширование одной строки с выбором алгоритма.
   - Сравнение двух строк по их хешам.
   - Хеширование файла (с измерением времени обработки).
   - Хеширование списка строк и вывод результатов в виде таблицы.

4. **Обработка ошибок**
   - Сообщения об ошибке при неверном выборе пункта меню.
   - Сообщения об ошибке, если файл не найден или алгоритм не поддерживается.

## 5. Описание модулей

### 1. `core.py` — ядро системы

Содержит чистые функции и классы для работы с модулем `hashlib`:

- `get_hasher(algo)` — создать объект хеш‑функции по названию алгоритма.
- `hash_text(text, algo)` — посчитать хеш строки (возвращает `hexdigest`).
- `hash_file(path, algo)` — посчитать хеш файла (чтение по частям).
- `compare_texts(text1, text2, algo)` — сравнить две строки по их хешам.
- `hash_many_texts(texts, algo)` — посчитать хеши для набора строк.
- `FileHashInfo` — датакласс с информацией о файле и его хеше.
- `get_file_hash_info(path, algo)` — получить расширенную информацию о хеше файла.

### 2. `utils.py` — утилиты

Содержит вспомогательные функции:

- `clear_screen()` — очистка экрана консоли.
- `separator()` и `print_title()` — красивые разделители и заголовки.
- `timed(func)` — декоратор для измерения времени выполнения функций.
- `ALGO_DESCRIPTIONS` и `describe_algorithm(name)` — краткие описания алгоритмов (`md5`, `sha1`, `sha256`, `sha512`).

### 3. `logic.py` — логика демонстраций

Содержит функции‑демонстрации и теоретические блоки:

- Теория:
  - `show_theory_intro()` — введение в хеш‑функции и модуль `hashlib`;
  - `show_theory_algorithms()` — обзор основных алгоритмов;
  - `show_theory_digest_vs_hexdigest()` — объяснение `digest()` и `hexdigest()`;
  - `show_theory_security_notes()` — замечания по безопасности.
- Практика:
  - `demo_hash_string()` — хеширование одной строки;
  - `demo_compare_strings()` — сравнение двух строк по хешу;
  - `demo_hash_file()` — хеширование файла с таймером;
  - `demo_many_hashes()` — хеширование нескольких строк и вывод таблицы.

### 4. `app.py` — главное приложение

Содержит:

- `show_menu()` — функция отображения главного меню.
- `MENU_ACTIONS` — словарь соответствия номера меню и функции.
- `main()` — главный цикл программы: чтение выбора пользователя, запуск нужной демонстрации, выход по `0`.

## 6. Как запустить проект

### Требования

- Python 3.8 и выше
- Установленный Git (для работы с репозиторием, опционально)

### Шаги запуска

1. Склонировать или скачать проект.
2. Перейти в папку проекта:

   ```bash
   cd path/to/hashlib-demo
   ```

3. Запустить приложение:

   ```bash
   python app.py
   ```

4. В консоли появится текстовое меню.  
   Выберите нужный пункт, введя цифру от `0` до `8`, и нажмите Enter.

## 7. Демонстрационные сценарии

1. **Теория**  
   Пользователь может в удобном виде прочитать основные сведения о хешах, алгоритмах, методах `digest`/`hexdigest` и ограничениях по безопасности.

2. **Хеширование строки**  
   Пользователь вводит строку и выбирает алгоритм (`md5`, `sha1`, `sha256`, `sha512`).  
   Программа выводит шестнадцатеричный хеш.

3. **Сравнение строк по хешу**  
   Пользователь вводит две строки, выбирает алгоритм.  
   Программа показывает их хеши и делает вывод: совпадают или нет.

4. **Хеширование файла**  
   Пользователь указывает путь к файлу и алгоритм.  
   Программа считает хеш файла, выводит размер, алгоритм и значение хеша, а также время обработки.

5. **Хеширование нескольких строк**  
   Пользователь вводит несколько строк подряд.  
   Программа выводит таблицу вида: номер, строка, её хеш.
